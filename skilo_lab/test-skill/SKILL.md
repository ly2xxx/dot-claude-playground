# Test Skill

This is a basic test skill to verify Skilo's validation and formatting features.

## Usage

1. Validate this skill: `skilo validate test-skill/`
2. Format this skill: `skilo fmt test-skill/SKILL.md`
3. Check formatting: `skilo fmt --check test-skill/SKILL.md`

## Expected Features

- YAML frontmatter validation
- Description completeness check
- Line count monitoring
- Naming convention enforcement
